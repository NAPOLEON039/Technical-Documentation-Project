Type selectors

HTML elements are written using a tag with some attributes like "class" and "id" which are quite common for nearly all elements. We can say that the type of an element is denoted by it's tag. Here are a few examples

  1. <a href="#">Follow on Twitter<a> <a href="#">Send an email<a> <a href="#">Follow on Reddit<a>

    Now to select all the links above, we need to target them using their type. We can do this using the tag.

    a {
      background-color: green;
    }

    All links will get a background color of green since we have targeted them by type.

  2. <p>This is a paragraph<p>
    <div>This is a div<div>

    Here, if we use the type selector "p", it will only target the first element and not the second, because they are both of different types.

first-child and last-child

:first-child and :last-child are pseudo-classes that make it posible to select the first and last child of a particular element. So if an element has 3 child elements, we can use :first-child to select only the very first child from the 3 child elements.

Similarly, using :last-child will select the very last child element from those 3 elements. Here is an example:

HTML:
<ul>
  <li> Number 1 <li>
  <li> Number 2 <li>
  <li> Number 3 <li>
  <li> Number 4 <li>
  <li> Number 5 <li>
<ul>
CSS:
li:first-child {
  color: red; }

In this example we apply the pseudo-class :first-child to a list element. The result would be:

  • Number 1
  • Number 2
  • Number 3
  • Number 4
  • Number 5

not pseudo-class

The :not() pseudo-class is perhaps the most powerful of all pseudo-classes. It returns all elements that do not match the selection.

For example, we need to select link elements but leave out those that have the class of "visited". In that case, the CSS would be something like this:

a:not(.visited) {
  color: green;
}

This will target all the links that do not have the class of "visited" and give them a font color of green
